   Weekly Task Report
==================

This week, I developed Python classes to handle spatial data parsing for my Geomatics Engineering assignment. 
Specifically, I created ``Point`` and ``Line`` classes using Object-Oriented Programming (OOP) principles. 
These classes can read custom-formatted text files containing coordinates and geometry names, and calculate the total geometric length of the lines.
